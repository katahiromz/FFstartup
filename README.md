# FFstartup

## What's this?

This is a command-line program to modify the startup entries for Windows.

## Usage

```txt
Usage #1: FFstartup -add ENTRY_NAME CMDLINE...
Usage #2: FFstartup -remove ENTRY_NAME
Usage #3: FFstartup -help
Usage #4: FFstartup -version
```
